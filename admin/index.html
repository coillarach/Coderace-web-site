Simple SELECT statements

Find the answer to these questions by typing a SELECT statement into the box on the left. Click SUBMIT and the results will appear in the box on the right.

You will be using these two tables:

  • COUNTRY ( name, code, capital, province, area, population )
  • CITY ( name, country, province, population, longitude, latitude )

1. Show all countries smaller than 50 square kilometres from the COUNTRY table
2. Show the country code for Greece from the COUNTRY table
3. Use the country code from the last question to find the names of all the cities in Greece from the CITY table
